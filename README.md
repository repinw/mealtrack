# mealtrack

Die Anwendung wird ermöglichen Kalorien und Nährwerte von Nahrung durch manuelle Eingabe oder durch Barcode aufzuschreiben und zu verfolgen.

Desweiteren wird eine Funktion folgen, die es ermöglicht einen "digitalen Kühlschrank" zu führen und die dort aufgezeichnete Nahrung in die Kalorienberechnung einzubeziehen.

Digitaler Kühlschrank:
- Einfügen und Entfernen von Nahrung
- Anzeige von Kosten
- Scan vom Digitalen Kassenbon
- Vorschläge für Einkauf (Einkaufsliste)

Weitere Ideen:
- Mahlzeiten planen mit dem Inhalt vom digitalen Kühlschrank
- Kosten und Nährwerte von geplanter Mahlzeit anzeigen
- Durch Eintrag in die Kalorienberechnung die Möglichkkeit haben Nahrung aus dem digitalen Kühlschrank zu streichen
