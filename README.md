# mealtrack

Die Anwendung wird ermöglichen Kalorien und Nährwerte von Nahrung durch manuelle Eingabe oder durch Barcode aufzuschreiben und zu verfolgen.

Desweiteren wird eine Funktion folgen, die es ermöglicht einen "digitalen Kühlschrank" zu führen und die dort aufgezeichnete Nahrung in die Kalorienberechnung einzubeziehen.
